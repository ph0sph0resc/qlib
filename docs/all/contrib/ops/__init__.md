# qlib.contrib.ops.__init__

## 模块概述

`qlib.contrib.ops.__init__` 模块是 Qlib 高频操作符的初始化模块。该模块目前为空，未导出任何公共接口。

## 说明

该模块主要用于作为 `qlib.contrib.ops` 包的标记文件。高频相关的操作符实现在 `high_freq.py` 文件中定义。

## 相关模块

- `qlib.contrib.ops.high_freq` - 提供高频数据处理操作符

## 使用示例

```python
# 高频操作符从 high_freq 模块导入
from qlib.contrib.ops.high_freq import DayCumsum, DayLast, FFillNan
```
